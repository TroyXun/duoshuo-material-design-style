# Duoshuo Style of Material Design
> 版本：1.2.0

## Introduction

风格参考 [C7V5](http://c7sky.com/wordpress-theme-c7v5.html) 。

+ `扁平化`
+ `阴影化`
+ `圆角化`

## 食用方法

+ 将多说的发布按钮的文字更改为：`send`
+ 将多说的正在发布的文字更改为：`hourglass_empty`
+ 将多说的默认头像换成：`https://avatar.duoshuo.com/avatar-100/212/266416.jpg`
